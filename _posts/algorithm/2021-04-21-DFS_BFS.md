그래프를 탐색하는 방법으로 `깊이우선 탐색` 과 `너비우선 탐색` 이 있습니다. 알고리즘 문제를 풀면서 개념을 다시한번 정리하고자 작성했습니다. `자바스크립트`를 사용하는 방식입니다.

## 깊이우선 탐색 (DFS : Depth First Search)

- 루트 노드에서 시작하여 최대한 깊이 내려간 후 다음 분기(branch)로 이동하는 탐색 방법입니다.
- 검색속도 자체는 너비우선 탐색보다 느립니다.
- 너비우선 탐색보다 구현 방식이 간단합니다.
- 모든 노드를 방문하고자 할 때 적합한 방식입니다.

→ **하나의 스택과 하나의 큐로 구현합니다.**

→ **어떤 노드를 방문했는지 여부를 검사해야 합니다**.(무한루프 방지)(큐에 저장)

## 너비우선 탐색 (BFS : Breadth First Search)

- 루트 노드에서 시작하여 인접한 노드부터 탐색하는 방법입니다.
- 주로 두 노드 사이의 최단거리를 찾을 때 적합한 방식입니다. (가까운 노드부터 탐색)

→ 두개의 **큐로 구현합니다.**

→ **어떤 노드를 방문했는지 여부를 검사해야 합니다**.(무한루프 방지)(큐에 저장)

## 시간복잡도

- 두 방식 모두 모든 노드를 방문한다는 점에서 시간 복잡도는 동일합니다.
- 그래프를 구현하는 방식에서 차이가 있습니다. (**인접 행렬 vs 인접리스트**)
- 인접 리스트 : O(N+E) (N : 노드 , E : 간선)
- 인접 행렬 : O(N^2)

→ **일반적으로 간선의 크기가 N^2보다 상대적으로 작기 때문에 인접 리스트로 구현하는 것이 더 효율적입니다.**

## JS를 사용하여 구현한 예시

### DFS - 하나의 큐와 하나의 스택

```jsx
const graph = {
  A: ["B", "C"],
  B: ["A", "D"],
  C: ["A", "G", "H", "I"],
  D: ["B", "E", "F"],
  E: ["D"],
  F: ["D"],
  G: ["C"],
  H: ["C"],
  I: ["C", "J"],
  J: ["I"],
};

// (graph, 시작 정점)
const dfs = (graph, startNode) => {
  let needVisitStack = []; // 탐색을 해야 할 노드들
  let visitedQueue = []; // 탐색을 마친 노드들

  needVisitStack.push(startNode);

  // 탐색을 해야 할 노드가 남아 있다면
  while (needVisitStack.length !== 0) {
    const node = needVisitStack.pop();
    if (!visitedQueue.includes(node)) {
      visitedQueue.push(node);
      needVisitStack = [...needVisitStack, ...graph[node]];
    }
  }

  return visitedQueue;
};

console.log(dfs(graph, "A"));

// ["A", "C", "I", "J", "H", "G", "B", "D", "F", "E"]
```

### BFS - 두개의 큐

```jsx
const graph = {
  A: ["B", "C"],
  B: ["A", "D"],
  C: ["A", "G", "H", "I"],
  D: ["B", "E", "F"],
  E: ["D"],
  F: ["D"],
  G: ["C"],
  H: ["C"],
  I: ["C", "J"],
  J: ["I"],
};

const bfs = (graph, startNode) => {
  let visited = []; // 탐색을 마친 노드들
  let needVisit = []; // 탐색해야할 노드들

  needVisit.push(startNode); // 노드 탐색 시작

  while (needVisit.length !== 0) {
    // 탐색해야할 노드가 남아있다면
    const node = needVisit.shift(); // queue이기 때문에 선입선출, shift()를 사용한다.
    if (!visited.includes(node)) {
      // 해당 노드가 탐색된 적 없다면
      visited.push(node);
      needVisit = [...needVisit, ...graph[node]];
    }
  }
  return visited;
};

console.log(bfs(graph, "A"));
// ["A", "B", "C", "D", "G", "H", "I", "E", "F", "J"]
```
