# The Problem

react에 `svg image`를 적용하는 도중 `<img />`와 같이 사용할 수 없는 문제가 발생했다. 보통 svg 아이콘을 쉽게 사용하게 해주는 material 라이브러리 등을 사용해왔지만, 직접 제작한 로고를 로컬에 함께 저장해 놓고 사용하고자 하여 방법을 찾게 되었다.

# **The Explanation**

### svg image란?

- Scalable Vector Graphics
- 벡터와 같은 2차원그래픽용 XML기반의 형식

### svg 사용 이유

- 확대 시 깨짐이 없다.
- 용량이 적다.
- 출력이 빠르다.
- 수정이 가능하다.
- 애니메이션이 가능하다.

# **The Solution**

### 1. 직접 `<svg>` Component 생성하기

```jsx
const svg = {
  edit: "M12 20h9M16.5 3.5a2.121 2.121 0 013 3L7 19l-4 1 1-4L16.5 3.5z",
};

const SVG = (props) => {
  return (
    <svg
      x="0px"
      y="0px"
      width={props.width ? props.width : "auto"}
      height={props.height ? props.height : "auto"}
      viewBox="0 0 24 24"
      stroke="currentColor"
      fill={props.color}
    >
      <path
        // fill={props.color}
        d={svg[props.name]}
      />
    </svg>
  );
};

// component로의 사용
<SVG name="edit" width="20px" height="20px" color="#aaa"></SVG>;
```

장점 : custom및 확장에 있어 용이하다.
단점 : svg와 react가 낯설다면 진입장벽이 존재한다.

### 2. `ReactComponent` 이용하기

```jsx
import { ReactComponent as MySvg } from "../../Images/Basic/mysvg.svg";
<MySvg />;

// 그리고 styled component를 쓴다면
const StyledMyIcon = styled(MySvg)`
  color: white;
`;
```

장점 : 간단하게 `import` 하여 Component화 시켜줄 수 있다. `styled-component` 를 사용하면 custom도 쉽다.

단점 : `create-react-app`을 통해서 만든 경우에만 동작한다.

→ create-react-app을 통해 생성했으므로 2번째 방법을 사용하여 컴포넌트화 했다.
